# 🔐 Secure Network Implementation using VPN, Firewall and IPS

## 📌 Project Overview
This project demonstrates the design and implementation of a **secure network architecture** using **VPN, Firewall, and Intrusion Prevention System (IPS)** concepts.  

The goal is to protect an organization's internal network from external threats while enabling **secure remote access** and **controlled network traffic**.

---

## 🎯 Objectives
- Design a **secure network topology**
- Implement **VPN for encrypted remote access**
- Configure **Firewall rules for traffic filtering**
- Simulate **Intrusion Prevention System (IPS)** using Access Control Lists (ACL)
- Ensure **Confidentiality, Integrity, and Availability (CIA Triad)**

---

## 🛠 Technologies Used
- Cisco Packet Tracer
- Router Configuration
- Firewall Security Policies
- Access Control Lists (ACL)
- Network Security Concepts

---

## 🖥 Network Architecture
The network consists of:

- **Router** – Acts as VPN Gateway  
- **Firewall** – Controls and filters network traffic  
- **Internal Network** – Company PCs and Server  
- **External Network** – Simulated Attacker System  
- **IPS Simulation** – Implemented using ACL rules  

---

## ⚙ Implementation Steps
1. Designed network topology in **Cisco Packet Tracer**
2. Configured **IP addresses for router and devices**
3. Implemented **Firewall for traffic filtering**
4. Created **ACL rules to block attacker IP**
5. Simulated **VPN gateway functionality**
6. Tested network communication and security

---

## 🔍 Testing
Network security was verified using **ping tests**.

### Internal Network Test
PC1 → Server communication tested successfully.

### Router Connectivity Test
PC1 pinged router gateway to confirm internal connectivity.

### Security Test
Attacker PC attempted to access internal network.

**Result:**  
❌ Access blocked by firewall and ACL rules.

---

## 📊 Results
- Unauthorized access attempts successfully blocked
- Secure internal communication maintained
- Firewall and ACL provided effective protection
- VPN gateway ensured secure remote connectivity
