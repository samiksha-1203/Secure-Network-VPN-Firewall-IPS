# Secure Network Implementation using VPN, Firewall and IPS

## Overview
This project demonstrates a secure network architecture using VPN, Firewall and Intrusion Prevention System (IPS).

## Objectives
- Implement VPN for secure remote access
- Configure firewall rules to control network traffic
- Use ACL to simulate IPS for attack prevention

## Tools Used
- Cisco Packet Tracer
- Router
- Firewall
- ACL

## Testing
- Internal PC to Server communication tested
- Router connectivity verified
- Attacker PC blocked by firewall

## Result
The firewall and ACL successfully prevent unauthorized access while allowing secure communication inside the network.
